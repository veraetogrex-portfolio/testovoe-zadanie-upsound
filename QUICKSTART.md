# Быстрый старт

## 1. Подготовка окружения

```bash
# Создайте виртуальное окружение
python -m venv venv

# Активируйте его (Windows)
venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

## 2. Настройка токена

Создайте файл `.env`:

```bash
copy .env.example .env
```

Откройте `.env` и добавьте токен от [@BotFather](https://t.me/BotFather):

```
BOT_TOKEN=ваш_токен_здесь
LOG_LEVEL=INFO
```

## 3. Запуск

```bash
python run.py
```

## 4. Тестирование

Откройте Telegram, найдите вашего бота и отправьте:

```
/start
```

Затем отправьте ссылку на трек:

```
https://music.yandex.ru/album/3192570/track/26464125
```

Бот вернет информацию о треке!

## Остановка

Нажмите `Ctrl+C` в консоли.

## Решение проблем

### Ошибка "BOT_TOKEN не найден"
- Проверьте, что файл `.env` создан
- Проверьте, что токен правильно скопирован без пробелов

### Ошибка при установке зависимостей
- Убедитесь, что у вас Python 3.10+
- Попробуйте обновить pip: `python -m pip install --upgrade pip`

### Бот не отвечает
- Проверьте логи в консоли
- Убедитесь, что токен валидный
- Проверьте интернет-соединение
